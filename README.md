# Healthcare Newsletter Platform

A personalized newsletter platform for healthcare providers to deliver targeted health information to patients.

## Features

- Patient management with health condition tracking
- Newsletter template creation with personalization
- Campaign management and scheduling
- Analytics and health outcome correlation
- HIPAA-compliant email delivery
- EHR integration capabilities

## Getting Started

1. Clone this repository
2. Install dependencies: `npm install`
3. Set up environment variables (see `.env.example`)
4. Run development server: `npm run dev`
5. Open http://localhost:3000
